Site Information

This site is currently under development.

The goal of this website is to incorporate some of HTML and CSS projects from the CoderDojo and Raspberry Pi projects page.

  • The website code is stored in a Github reposistory.
  • Everytime our main website reposistory is updated, our website is built and deployed to Vercel (a web hosting company with a very good free plan)
  • The mentor form uses an AWS API endpoint to run an AWS Lambda function to email the hello@carrickdojo.com address and the address supplied by the user. While the AWS hosted part is quite an advanced concept, the API utilizes a standard HTML post to submit the data.
