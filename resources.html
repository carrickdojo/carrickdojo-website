There are lots of super resources on the Internet. Here are some of our favourites.

What else should be on our list?

Just like a big city, the World Wide Web is a place of countless people. There are lots of safe and cool places to visit, but also a few dangers that we need to learn about.

These websites are designed for people of all ages and skill levels.

Scratch is the world's largest coding community for children and a coding language with a simple visual interface.

The Raspberry Pi Foundation have step-by-step instructions to teach you how to create games, animations and much more.

The CoderDojo have their own section on the Raspberry Pi Projects page with super projects, different paths and great ideas.

The Raspberry Pi CodeEditor for creating HTML/CSS or Python projects.

This is new and currently in development, however is aimed at people of all ages starting out.

Microsoft's MakeCode editor is the perfect way to start programming and get creating with the BBC micro:bit.

The colour-coded blocks are familiar to anyone who's used Scratch and can access all the features of the micro:bit.

Some Key Open Source Tools and Applications

CoderDojo supports the open source community and all the amazing tools that people generously create for everyone to share.

Before installing any software, you should think about what you want the software to do and check the software is trustworthy.

This is a tiny list of some key applications, it is highly subjective and there are countless amazing omissions.

What else should be on this list?

Blender

A free and open-source 3D graphics software tool set used for creating animated films, visual effects, art, 3D-printed models, motion graphics, interactive 3D applications, virtual reality, and video games.

Gimp

Gimp (GNU Image Manipulation Program) is the free & open source image editor

Krita

A professional FREE and open source painting program. It is made by artists that want to see affordable art tools for everyone.

Inkscape

Whether you are an illustrator, designer, web designer or just someone who needs to create some vector imagery, Inkscape is for you!

Godot

The Godot Engine is a free, all-in-one, cross-platform game engine that makes it easy for you to create 2D and 3D games.

OBS Studio

Free and open source software for video recording and live streaming.

Firefox

An open source media player than can play virtually anything.

LibreOffice

LibreOffice is a free and powerful office suite, and a successor to OpenOffice

Git

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.
